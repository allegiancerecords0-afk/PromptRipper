This script will rip your positive prompt from each of the images in a folder of images that have workflow metadata and populate a text file in that directory. It will automatically ignore duplicates. I currently have it set up so simple single sampler workflows it will give the sampler and scheduler and amount of steps used in that workflow as well. In the future I'm thinking maybe a full dump of all the relevant settings including for MoE setups where there's multiple samplers and what LoRAs were used with strength etc

I think it only works with ComfyUI workflows but not sure on other UI metadata structuring.
